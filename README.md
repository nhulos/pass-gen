# Pass Gen
## You can print 
* name
* username
* phone number
* date(dd-mm-yyy)
* other

Print `gen!` for generate combinations.

You can look generated combinations in pass.txt.

## Installation
If you havent `git` or `python`, you need install it.
```
git clone https://github.com/nhulos/pass-gen/edit/main
cd pass-gen
chmod +x pass_gen.py
./pass_gen.py
```
